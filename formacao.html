Formação um para um

Formação individual ou para grupos

O que é?

Formação para uma ou mais pessoas.

Esta aprendizagem dedicada permite obter melhores resultados e num curto espaço de tempo podendo ser ministrada nas nossas instalções ou ao domicílio.

Vantagens

  • Conteúdos adaptados às necessidades do Formando;
  • Rapidez de aprendizagem (1 terço das horas de um curso tradicional);
  • Horário Flexível;
  • Certificado de frequência;

A quem se destina?

A todas as pessoas interessadas em efetuar uma aprendizagem rápida em ambiente personalizado.

Cursos

PROGRAMAÇÃO PARA ANDROID

No final do curso o participante estará apto a desenvolver aplicações para a plataforma Android usando o IntelliJ Idea e o Android Sdk

PROGRAMAÇÃO AVANÇADA PARA ANDROID

No final do curso o participante terá uma aplicação feita por si utilizando conceitos avançados de Android.

HTML5, JAVASCRIPT, JQUERY E CSS3

No final do curso o participante estará apto a criar Sites em Html5 acrescentando interatividade através da linguagem JavaScript e da biblioteca Jquery e usando formatação Css3.

PROGRAMAÇÃO WEB COM PHP E MYSQL

No final do curso o participante estará apto a desenvolver Web Sites dinâmicos usando a linguagem PHP e Base de Dados MySql.

Informações

Duração dos cursos: 2 meses, 2 vezes semana, 2h por sessão;

Horário: Segunda e Quarta das 18 às 20h (ou a combinar)

Certificado de frequência;
